# Claude Assistant - Chrome拡張機能

ページ情報を収集してClaude.aiに送信するためのChrome拡張機能です。**APIキー不要**で、既存のClaude.aiアカウントをそのまま使用できます。

## 特徴

- **APIキー不要**: Claude.aiに直接アクセスするため、APIキーの設定は必要ありません
- **ワンクリック操作**: ボタンを押すだけでページ情報をクリップボードにコピー
- **自動でClaude.aiを開く**: コピー後、自動的にClaude.aiの新規チャットを開きます

## 機能

### ページ分析
- **要約する**: ページの内容を簡潔に要約
- **解説する**: 内容を分かりやすく解説
- **翻訳する**: 日本語⇔英語の翻訳
- **質問を作成**: 理解を深めるための質問を生成

### ブラウザ情報
- **タブ一覧**: 開いているタブを整理するアドバイス
- **ブックマーク**: ブックマークの整理提案
- **閲覧履歴**: 興味関心の分析と提案

### カスタム質問
- 自由なテキストでClaudeに質問
- ページ内容を含めるかどうか選択可能

## インストール方法

### 1. アイコンを生成

1. `extension/icons/generate-icons.html` をブラウザで開く
2. 「全てのアイコンをダウンロード」をクリック
3. ダウンロードされた `icon16.png`, `icon48.png`, `icon128.png` を `extension/icons/` フォルダに移動

### 2. 拡張機能を読み込む

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `extension` フォルダを選択

## 使い方

1. 任意のWebページで拡張機能アイコンをクリック
2. 使いたい機能のボタンをクリック
3. プロンプトが自動的にクリップボードにコピーされる
4. Claude.aiが新しいタブで開く
5. Ctrl+V（またはCmd+V）で貼り付けて送信！

## ディレクトリ構造

```
extension/
├── manifest.json          # 拡張機能マニフェスト
├── background/
│   └── service-worker.js  # バックグラウンドサービス
├── popup/
│   ├── popup.html         # ポップアップUI
│   ├── popup.css          # スタイル
│   └── popup.js           # ポップアップロジック
├── content/
│   └── content.js         # コンテンツスクリプト
└── icons/
    ├── icon.svg           # SVGアイコン
    ├── generate-icons.html # アイコン生成ツール
    ├── icon16.png         # 16x16アイコン
    ├── icon48.png         # 48x48アイコン
    └── icon128.png        # 128x128アイコン
```

## 必要な権限

- `activeTab`: 現在のタブにアクセス
- `tabs`: タブ情報の取得
- `bookmarks`: ブックマーク情報の取得
- `history`: 履歴情報の取得
- `scripting`: ページコンテンツの抽出
- `clipboardWrite`: クリップボードへの書き込み

## プライバシー

- ページ情報はローカルで処理され、クリップボードにコピーされるだけです
- 外部サーバーへの自動送信は行いません
- Claude.aiへの送信はユーザー自身が貼り付けて行います
